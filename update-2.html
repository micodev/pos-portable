إضافة ملاحظات (Notes)

تم إضافة ميزة "الملاحظات" التي تسمح للموظفين بإضافة ملاحظات مرتبطة بعملية الشراء أو العميل أو المنتج. يمكن استخدام الملاحظات لتوثيق ملاحظات داخلية أو ملاحظات مرئية للعميل.

  • الحقول الأساسية: المؤلف (الموظف)، التاريخ/الوقت، نص الملاحظة، مرتبط بالمنتج/البيع/العميل.
  • خيار لتمييز الملاحظة كـ "داخلية" (لا يراها العميل) أو "مشاركة" (تكون مرئية للعميل).
  • عرض الملاحظات في صفحة المنتج، شاشة المبيعات، وتفاصيل العميل.

استخدامات شائعة مثال:

  • ملاحظة بخصوص حالة المنتج بعد الفحص (مثلاً: "الخدش طفيف" أو "يحتاج إعادة تغليف").
  • مذكرات خدمة العملاء (مثلاً: "عميل يخبرنا أنه يريد توصيلًا لاحقًا").
  • توثيق تغييرات يدوية في الفاتورة أو السياسات المطبقة.

السلعة التالفة (Broken Product)

أضفنا ميزة رفع ملف Excel/CSV لإدارة السلع التالفة (التالف). يقوم النظام بمعالجة الملف وتطبيق تحديث كميات التالف على المنتجات المطابقة بحسب النموذج.

تنسيق الملف المطلوب (أو CSV):

model,style,size,quantity
MDL-001,Classic,45,3
MDL-002,Modern,42,1
MDL-010,Classic,44,2

شرح الحقول:

  • model = رمز الطراز أو SKU
  • style = تصميم/نسخة المنتج
  • size = الحجم/قياسات المنتج
  • quantity = عدد الوحدات التالفة

التحقق سيتم كالتالي: البحث عن المنتج في النظام باستخدام (model, style, size) ومبدئياً عرض قائمة التطابقات، ثم تأكيد الإجراء لتخفيض الكمية ووسمها كـ "تالف".

موقع المتجر (Store Location)

أضفنا إمكانية رفع ملف لتعيين مواقع المنتجات داخل المتجر أو المخزن، بحيث يمكن عرض هذه المعلومات عند البحث عن المنتج. يمكن أن يساعد هذا في تقليل وقت البحث وتحسين تجربة الموظف والعميل.

تنسيق الملف المطلوب (أو CSV):

model,style,location
MDL-001,Classic,Aisle 3 - Shelf B
MDL-002,Modern,Back Room - Shelf 1
MDL-010,Classic,Front Display - Rack 2

التحقق: يعرض النظام التطابقات بناءً على (model, style) ويتيح للمستخدم مراجعة وتأكيد تعيين الموقع قبل حفظه.
